Help

The Streamer interface is designed to help us easily change the information displayed on the atteched wallscreen. This interface supports API commands directly or you can simply use the selector on the control panel.

The control panel has been configured to display a set of pre-defined streams that should be present (coming from dedicated VMs to stream content). If you would like additional values added to the interface, you will need to open a ticket or have someone edit the control.html file on this device.

However, if you are a little more adventurous, simply stream content to the 'streamer' host and use the API to modify the wallscreen to display the streamed content. The following commnad can be used to pull the streamed content and display it on this device.

http://ip of this device/control?server=stream server&application=application name&stream=stream

If you choose to leave any of the parameters out of the request, the following defualts will be used..or it will simply use the current value (from the previous request).

  • server = stream.ihire.local
  • application = live
  • stream = ...as defined in the displayscreenWS.py file.
